### Features

- Anonymous attack through TOR

### Usage:

```
git clone https://github.com/thelinuxchoice/facebash
cd instashell
chmod +x facebash.sh
service tor start
sudo ./facebash.sh
```

### Install requirements (Curl, Tor):

```
chmod +x install.sh
sudo ./install.sh
```
